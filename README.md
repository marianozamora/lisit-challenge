## Getting Started
project setup
```bash
pnpm install
```

run tests
```bash
pnpm test
```

run dev server
```bash
pnpm dev
```

build for production
```bash
pnpm build
```


### Prerequisites

- [Node.js](https://nodejs.org/en/) - v14.17.0
- [pnpm](https://pnpm.io/) - v6.14.5
- [Vite](https://vitejs.dev/) - v2.4.4
- [React](https://reactjs.org/) - v17.0.2
- [TypeScript](https://www.typescriptlang.org/) - v4.3.5
